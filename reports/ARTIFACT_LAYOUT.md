# Experiment Artifact Layout

New experiment artifacts should be archived under:

`outputs/yy-mm-dd-hh-min-test-name/`

Each archive folder should contain:

- `<test-name>_report.md`
- `programs/`
- `outputs/`
- `MANIFEST.md`
